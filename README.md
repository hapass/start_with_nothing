# Glow

## Build

1. Install [brew](https://brew.sh)
1. Run. Terminal: *brew install haxe*
1. Run. Terminal: *./build.sh release*

## Run

1. Launch *index.html* or [play here](https://hapass.github.io/start_with_nothing/)
